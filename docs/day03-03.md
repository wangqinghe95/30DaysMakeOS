# Day03-03

1. 验证读取磁盘是否成功

Makefile 中对第二个磁盘写入数据

```
haribote.img: ipl.bin
	dd if=/dev/zero of=haribote.img bs=512 count=2880
	dd if=ipl.bin of=haribote.img bs=512 count=1 conv=notrunc
	printf "HELLO_KERNEL" | dd of=haribote.img bs=512 seek=1 count=1 conv=notrunc
```

启动 qemu 监控模式

```
run-debug:
	qemu-system-x86_64 -drive file=haribote.img,format=raw,if=floppy -boot a -monitor stdio
```

然后在qemu的终端中查看 0x8200 内存处的数据，输出如下表示成功

```
ubuntu22@ubuntu22:~/30DaysMakeOS$ qemu-system-x86_64 -drive file=haribote.img,format=raw,if=floppy -boot a -monitor stdio
QEMU 6.2.0 monitor - type 'help' for more information
(qemu) xp /16bx 0x8200
0000000000008200: 0x48 0x45 0x4c 0x4c 0x4f 0x5f 0x4b 0x45
0000000000008208: 0x52 0x4e 0x45 0x4c 0x00 0x00 0x00 0x00
```
