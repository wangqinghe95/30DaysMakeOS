# Day02

## 1. 计算机启动流程

1. 计算机启动
   ↓
2. BIOS 读取磁盘第 1 扇区（512 字节）到内存 0x7C00
   ↓
3. BIOS 检查 0x7DFE 处是否为 0x55 0xAA
   ↓ (是)
4. BIOS 跳转到 0x7C00 开始执行
   ↓
5. 遇到 `JMP entry`，跳转到真正的代码入口
   ↓
6. 初始化段寄存器和栈（SS=0, SP=0x7C00）
   ↓
7. 循环读取 `msg` 字符串，调用 `INT 0x10` 输出每个字符
   ↓
8. 遇到字符串结束符 `0`，退出循环
   ↓
9. 执行 `HLT` + `JMP fin`，进入无限停机状态

## 2. 二进制文件结构示意图

偏移地址   内容
──────── ──────────────────────────────────────────────
0x0000   EB 4E 90 48 45 4C 4C 4F ...  ← FAT12 文件头
...      
0x003E   00 00 00 00 00 00 00 00 ...  ← RESB 18 填充
0x0050   B8 00 00 8E D0 BC 00 7C ...  ← entry: 代码开始
0x005A   BE 26 7C AC 83 C6 01 3C ...  ← 显示循环
0x0074   F4 EB FE                      ← fin: HLT/JMP
0x0077   0A 0A 68 65 6C 6C 6F 2C ...  ← msg: 字符串
0x0087   00                            ← 字符串结束符
...     （RESB 填充 0，直到 0x1FD）
0x1FE   55 AA                         ← 引导标志
0x1FF   （文件结束，恰好 512 字节）
